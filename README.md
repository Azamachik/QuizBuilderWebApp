## О проекте

**ТЕСТО** — веб-приложение для создания, публикации и прохождения тестов. Платформа позволяет авторам собирать тесты с вопросами разных типов и делиться ими по инвайт-ссылке, а участникам — проходить тесты без регистрации и сразу видеть результаты.

### Задача

Упростить процесс создания образовательных и развлекательных тестов: убрать необходимость использовать громоздкие инструменты (Google Forms, сторонние сервисы) и предоставить удобный, минималистичный интерфейс с мгновенным деплоем тестов.

### Целевая аудитория

- **Преподаватели и тренеры** — для проверки знаний студентов и участников курсов
- **HR-специалисты** — для проведения входного тестирования кандидатов
- **Контент-мейкеры** — для создания развлекательных викторин
- **Разработчики и дизайнеры** — для технических квизов внутри команды

## Возможности

| Функция | Описание |
|---------|----------|
| Регистрация и вход | Email/пароль |
| Управление тестами | Создание, редактирование, удаление, публикация / черновик |

| Результаты | Просмотр итогов прохождения с правильными ответами |
| Переключение вида | Отображение тестов плитками или списком |
| Профиль пользователя | Редактирование данных, аватар, статистика |
| График активности | Heatmap прохождений за последний год |
| Тёмная/светлая тема | Переключатель в шапке сайта |
| Адаптивный UI | Отдельные мобильные варианты ключевых страниц |

## Архитектура — Feature Sliced Design

```
src/
├── app/               # Инициализация, провайдеры (Store, Theme, Router), глобальные стили
├── pages/             # Страницы приложения (lazy-loaded)
│   ├── LandingPage        # Главная для гостей
│   ├── LoginPage          # Вход
│   ├── RegisterPage       # Регистрация
│   ├── QuizzesPage        # Список тестов авторизованного пользователя
│   ├── QuizEditorPage     # Редактор теста и вопросов
│   ├── QuizTakingPage     # Прохождение теста по токену
│   ├── QuizResultsPage    # Результаты прохождения
│   ├── ProfilePage        # Профиль пользователя
│   └── NotFoundPage       # 404
├── widgets/           # Крупные UI-блоки
│   └── Header             # Шапка сайта (desktop + mobile)
├── features/          # Пользовательские сценарии
│   ├── AuthByEmail        # Вход по email
│   ├── RegisterByEmail    # Регистрация по email
│   ├── CreateQuiz         # Создание теста
│   ├── EditQuiz           # Редактирование теста
│   ├── CreateQuestion     # Создание вопроса
│   ├── CreateLink         # Генерация инвайт-ссылки
│   ├── DragSort           # Сортировка вопросов drag-and-drop
│   └── ToggleTheme        # Переключатель темы
├── entities/          # Бизнес-сущности (Redux-слайсы, сервисы, UI)
│   ├── User               # Аутентификация
│   ├── Profile            # Профиль пользователя
│   ├── Quiz               # Тесты (QuizCard, QuizRow, StatusBadge)
│   ├── Question           # Вопросы (QuestionCard)
│   ├── Attempt            # Попытки прохождения
│   └── InviteLink         # Инвайт-токены
└── shared/            # Переиспользуемая база
    ├── ui/            # Button, Card, Dialog, NavLink, Skeleton, StatCard, Tooltip…
    ├── api/           # Axios-инстанс, RTK Query
    ├── config/        # routeConfig, Storybook decorators
    ├── consts/        # Константы (localStorage keys, theme)
    └── lib/           # Хуки (useAppDispatch, useAppSelector, useDynamicModuleLoader,
                       #        useIsMobile, useTheme), utils
```

## Запуск проекта

### Требования

- Node.js ≥ 18
- pnpm (рекомендуется) или npm

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd QuizBuilder/client
```

### 2. Установить зависимости

```bash
pnpm install
```

### 3. Запустить dev-сервер + mock API одной командой

```bash
pnpm start:dev
```

Или раздельно (два терминала):

```bash
# Терминал 1 — mock API (порт 3001)
pnpm json-server json-server/db.json --port 3001

# Терминал 2 — Vite dev-сервер
pnpm start
```

Приложение: **http://localhost:5173**  
Mock API: **http://localhost:3001**

### 4. Другие команды

```bash
pnpm build          # TypeScript-проверка + production-сборка
pnpm test           # Vitest с покрытием
pnpm lint:ts        # ESLint
pnpm lint:css       # Stylelint
pnpm storybook      # Storybook на http://localhost:6006
pnpm build-storybook # Статическая сборка Storybook
```

## Структура маршрутов

| Маршрут | Страница | Доступ |
|---------|----------|--------|
| `/` | Главная (LandingPage) | Только гости |
| `/login` | Вход | Только гости |
| `/register` | Регистрация | Только гости |
| `/quizzes` | Список тестов | Авторизованный |
| `/quizzes/:id` | Редактор теста | Авторизованный |
| `/profile` | Профиль пользователя | Авторизованный |
| `/quiz/:token` | Прохождение теста | Публичный |
| `/quiz/:token/results/:attemptId` | Результаты | Публичный |
| `*` | 404 | Публичный |
